# 清酒無双 構想メモ（プロトタイプ名: もろみ無双）

日本酒の醸造ロジックをハクスラ無双に昇華する。この文書が用語対応の正。

## 用語対応表

| 醸造 | ゲームシステム |
|---|---|
| 並行複発酵 | コア戦闘ループ。通常攻撃（櫂入れ）で「糖化（ブドウ糖）」を稼ぎ、消費して「発酵（アルコール必殺技）」を放つ |
| 精米歩合 | 武器強化。%を下げるほど攻撃力倍率が上がる（70→50→35、35未満なし） |
| 上槽 | クリア報酬。醪を搾って防具（酒米）がドロップするハクスラ判定 |
| 酒米（山田錦・五百万石…） | 防具。固定メインスキル＋ランダムサブスキル |
| 特定名称（純米/吟醸/大吟醸） | レア度。大吟醸=サブスキル4枠確定 |
| 酵母（きょうかい6号/7号/9号） | ブドウ糖を消費する固有チャージ技 |
| 麹（黄/白/黒） | 武器に塗る属性。並列の個性であり相性関係は持たせない |
| 火落ち菌 | 呂布枠。乱入中はゲージが減り続け、倒すまでクリア不可 |
| 酒蔵（Kura） | 拠点。初版は広大マップにせず1タンク=1蔵の連戦で進軍感を出す |
