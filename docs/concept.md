# 清酒無双 構想メモ（プロトタイプ名: もろみ無双）

日本酒の醸造ロジックをハクスラ無双に昇華する。この文書が用語対応の正。

## 用語対応表

| 醸造 | ゲームシステム |
|---|---|
| 並行複発酵 | コア戦闘ループ。通常攻撃（櫂入れ）で「糖化（ブドウ糖）」を稼ぎ、消費して「発酵（アルコール必殺技）」を放つ |
| 精米歩合 | 武器強化。%を下げるほど攻撃力倍率が上がる（70→50→35、35未満なし） |
| 上槽 | クリア報酬。醪を搾って防具（酒米）がドロップするハクスラ判定 |
| 酒米（山田錦・五百万石…） | 防具。固定メインスキル＋ランダムサブスキル |
| 特定名称（純米/吟醸/大吟醸） | レア度。大吟醸=サブスキル4枠確定 |
| 酵母（きょうかい6号/7号/9号） | ブドウ糖を消費する固有チャージ技 |
| 麹（黄/白/黒） | 武器に塗る属性。並列の個性であり相性関係は持たせない |
| 火落ち菌 | 呂布枠。乱入中はゲージが減り続け、倒すまでクリア不可 |
| 酒蔵（Kura） | 拠点。初版は広大マップにせず1タンク=1蔵の連戦で進軍感を出す |

## 数値確定候補（2026-07-19 sim/seishu_sim.py 実測・各1000回ロール）

- **糖チャージ率＝与ダメの2%**。原案の10%は毎ターン必殺が撃てて崩壊（1戦26発）。2%で1戦3〜5発の「必殺」感になる
- **酵母3種の役割分担**（三すくみなしでも並ぶ）:
  - きょうかい6号＝コスト80・全体3秒凍結 → 勝率99.8%・平均55.8t（最遅だが最安全。生存特化）
  - きょうかい7号＝コスト100・全体600%（原案500%から増） → 勝率97.2%・平均38.7t（殲滅速度特化・唯一事故る）
  - きょうかい9号＝コスト100・単体500%防御無視+2秒スタン（原案1000%から半減） → 勝率100%・平均38.2t（対ボス安定。強さの正体は倍率ではなくスタン）
- **精米歩合の式は原案どおり**: 70/50/35% → 攻撃力100/140/170。精米50→35は勝率0→100%を分ける支配的な強化で、成長の柱として機能する
- **上槽レア度テーブルは原案どおり**（NIGHTMARE=大吟醸100%を含め分布に問題なし）
- **要再設計（そのまま実装しないこと）**:
  - 箱麹BOX（攻撃範囲1.5倍）は罠装備。雑魚は素の一振りで刈り切れるため麹蓋LID（総ダメ+20%）一択になる。BOXは「ヒット数＝糖化効率」の強みに設計し直す
  - 黄麹（クリダメ+20%）は3麹中最弱（期待値+7%程度）。白麹（被ダメ-20%）が汎用最強
  - 山田錦メイン（クリ率+25%）は五百万石（速度+20%）に勝率で劣る。クリティカル軸全体の底上げが必要
- モデルの前提: 1tick=1秒・回避50%でアクション層を近似した比較用の土俵。絶対値でなく差を読む
