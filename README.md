# 清酒無双 🍶

**醸して、なぎ倒せ。** 日本酒の醸造工程をそのままゲームシステムに昇華した、ブラウザで遊べる醸造ハクスラ無双。

## ▶ 遊ぶ

**https://y-kanno-sakenote.github.io/seishu-musou/**

インストール不要・単一HTML。PC（キーボード）とスマホ（タッチ・横持ち推奨）に対応。

## どんなゲーム？

雑菌に冒された各地の蔵を、麹坊やが櫂棒一本で浄化して回ります。

- **並行複発酵バトル**：櫂入れで雑菌を薙ぎ払うと「糖」が溜まり（糖化）、消費して必殺技（発酵）を放つ
- **火落ち菌の乱入**：蔵の発酵度70%で警報とともに乱入。殺菌するまで発酵度が下がり続ける呂布枠
- **米武将の浄化**：発酵度100%で「洗脳された山田錦」などの米武将が立ちはだかる。浄化すると同種の酒米が確定ドロップ
- **ハクスラ装備**：酒米（防具・並/純米/吟醸/大吟醸）と麹武器（黄/白/黒×箱麹/麹蓋）を集めて持ち替え
- **成長**：精米歩合（70→50→35%）、酵母3種の必殺技（7号=全体爆発／9号=一点突破＋スタン／6号=凍結）
- **進軍マップ**：先鋒の蔵→分岐（泥蔵/寒蔵）→総本山の3戦で地方制圧。死んでも装備は持ち越し

## 操作

| | PC | スマホ |
|---|---|---|
| 移動 | WASD / 矢印 | 左側をドラッグ |
| 櫂入れ | Space | 自動 |
| 必殺（発酵） | X | 右下ボタン |
| 装備画面 | E（進軍図で） | 装備ボタン |
| 酵母切替 | C（進軍図で） | 装備画面から |
| 音オン/オフ | M | − |
